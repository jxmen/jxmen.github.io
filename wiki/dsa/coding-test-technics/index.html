Heap

from heapq import heappush

heap = []
heapq.heappush(heap, 50) # 최소 힙에 값 넣기

최대 힙은 부호를 변경하여 구현한다.

from heapq import heappush, heappop

heap = []
heapq.heappush(heap, -value) # 최대 힙

print(-heapq.heappop(heap))
